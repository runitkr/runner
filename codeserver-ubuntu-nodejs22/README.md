# run.it.kr - codeserver-ubuntu-nodejs22
```
Welcome to run.it.kr CodeServer on Ubuntu with Node.js 22!

This environment comes pre-configured with Node.js 22 installed via n-install.
You can enable additional package managers like yarn or pnpm by running:

  corepack enable yarn
  corepack enable pnpm

Some administrative permissions like sudo commands are
prohibited in this environment for security reasons.

If you need global npm packages, you can install them using without sudo, for example:

  npm install -g <package-name>

Happy coding!
```
